# Patched Fira Sans Webfonts

The patched fonts have a x-height matching according to Verdana which fixes a baseline bug which is visible when using Fira Sans in form Elements in Internet Explorer.

See bug description over here:
<https://github.com/mozilla/Fira/issues/57>

And this Blogpost which describes the problem:
<http://www.dead-pixel.de/align-texts-at-the-baseline-of-different-font-webfonts/>
